About

In this project, I will attempt to build a Django app that runs a survey website following the blueprint by Matthew Segal from his Blog Post.


On this website, users will be able to sign up to create surveys and share it with others. The surveys will have questions, and each question will have answers. Users, anonymously, will be able to fill the surveys with their responses. Users can fill out the survey infinitely. Should that be changed? Let me know.







These are features which are not implemented, yet.
  • * Password recovery
  • * Password reset
  • * Confirmation emails
  • * User information update
  • * User deletion
  • + Other stuff I can't think of right now.

If you need help with anything from the list please message me
